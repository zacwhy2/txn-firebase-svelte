## Creating Firebase app with Svelte

1. create project
2. create app

```sh
$ npm init -y
$ npm i -D firebase-tools
$ npx firebase login
$ npx firebase experiments:enable webframeworks
$ npx firebase init hosting

$ cd hosting
hosting $ npm i firebase
hosting $ npm run dev
```
